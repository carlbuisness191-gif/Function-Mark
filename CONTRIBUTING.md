# Contributing to Function Mark (FM)

We welcome contributions from theoretical physicists, mathematicians, and computational scientists! You can help advance the **Function Mark** framework in the following ways:

## How to Contribute
1. **Submit New Examples:** If you plug a broken physics equation into the FM template and successfully resolve its infinities, please add it to the `EXAMPLES.md` file.
2. **Improve Simulations:** Help optimize the Python scripts or build new simulation files in Wolfram Mathematica or MATLAB.
3. **Report Bugs:** If you find a specific mathematical configuration where the Heat-Kernel filter fails to converge, open an issue.

## Pull Request Process
* Fork the repository and create your branch from `main`.
* Ensure any Python code complies with standard formatting principles.
* Update documentation to clearly explain the mathematical physics behind your additions.
